# Odyssey Quest Library V1

## Quest Philosophy

Quest 不是题目。

Quest 是真实世界中的任务委托。

用户不是在学习。

用户是在完成委托的过程中证明自己的能力。

每个 Quest 对应：

* Skill
* Difficulty
* Assessment Dimension
* Deliverable

---
