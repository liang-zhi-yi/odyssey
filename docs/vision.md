# Odyssey Vision

## What is Odyssey

Odyssey 是一个基于能力图谱（Capability Graph）的 AI 成长操作系统。

它不关注用户学习了多少时间。

而关注用户真正掌握了什么能力。

通过：

* Path（成长路径）
* Quest（真实委托）
* Assessment（AI试炼）
* Progress（能力成长）
* Credential（能力认证）

帮助用户完成从目标到能力的转化。

---

## User Problem

当前学习产品主要记录：

* 学习时长
* 打卡次数
* 课程完成率

但无法回答：

* 我真的学会了吗？
* 我的能力处于什么水平？
* 下一步应该学什么？
* 如何证明自己的能力？
