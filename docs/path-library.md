# Odyssey Path Library V1

## Overview

Path（成长路径）是 Odyssey 中连接目标与能力的桥梁。

用户首先选择目标。

系统根据目标匹配对应成长路径。

每条成长路径由多个 Skill 组成。

用户通过完成 Quest、通过 Assessment、获得 Credential，逐步完成成长。

---

# Agent Engineer Path

## Path Description

目标：

成长为能够独立设计、开发、部署 AI Agent 系统的工程师。

完成本路径后，用户应具备：

* LLM调用能力
* Prompt设计能力
* Tool Calling能力
* RAG系统构建能力
* Workflow设计能力
* LangGraph开发能力
* Agent工程化能力

最终能够独立完成中小型AI Agent项目开发。
