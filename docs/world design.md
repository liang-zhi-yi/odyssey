# ODYSSEY 世界文明板块设计文档

## 一、设计目标

Odyssey 不再是传统学习平台，而是一个以「文明建设」为核心的成长世界。

用户学习的目的不再是提升技能数值，而是通过技能成长推动文明发展、建造建筑、解锁时代、获得成就与徽章，最终建立属于自己的知识文明。

核心成长逻辑：

技能学习 → 获得能力点 → 解锁建筑 → 提升文明等级 → 晋升时代 → 获得成就 → 建造文明奇观

用户在系统中扮演：

文明开拓者（Civilization Founder）

而非普通学习者。

# 八、重构要求

禁止：

* 传统后台仪表盘设计
* 纯数据统计展示
* 单纯技能等级系统

必须实现：

* 文明地图
* 建筑建造系统
* 文明时代系统
* 技能驱动建筑解锁
* 成就与徽章体系
* 建筑升级变化
* 文明成长时间轴
* 文明指数系统
* 世界探索机制

最终用户打开 Odyssey 时看到的应该是：

“我的文明世界正在成长”

而不是：

“我的技能分数提高了 5 分”。
